# Raw Sources

This directory stores local source material used as evidence for the PostgreSQL engine wiki.

## PostgreSQL Source Checkouts

PostgreSQL source checkouts live under:

```text
raw/postgres-NN/
```

For example:

```text
raw/postgres-18/
```

These checkouts are project-local dependencies. They should stay inside this repository directory but are ignored by git through `.gitignore`.

Durable source pins are recorded in `wiki/versions.md` and the matching `wiki/vNN/index.md` page.
